# environment

Tested with:

- bash 5.2.37(1)-release (x86_64-pc-linux-gnu)

This directory contains small, helpful tools for a consistent client workflow.  
They are not required for the functionality of the services themselves.

## Contents

Wrappers:

- `bcli`   → wrapper for bitcoin-cli
- `lcli`   → wrapper for lightning-cli
- `blog`   → bitcoind logs
- `llog`   → lightningd logs

Binary:

Binary:

- `hwi` → [Hardware Wallet Interface (HWI)](https://github.com/bitcoin-core/HWI) binary (external tool)

## HWI

This setup uses HWI v3.2.0  
provided by the official Hardware Wallet Interface project.
